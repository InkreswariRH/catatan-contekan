Object type adalah dimana class dijadikan sebagai tipe data. Sebagaimana kita ketahui bahwa ada berbagai macam tipe data, seperti integer, float, double, dan lain sebagainya. Nah object type ini merupakan situasi dimana class dijadikan sebagai tipe data dari sebuah variabel. Sehingga nantinya, hanya instance dari class tersebut saja yang akan dikenali oleh sistem. Selain itu, maka sistem akan eror.
